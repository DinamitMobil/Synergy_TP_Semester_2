# Кейс-задача № 1

Дан одномерный массив А размерности N. Найти сумму отрицательных элементов,
расположенных между максимальным и минимальным.
Ответом на задачу будет ссылка на репозиторий GitHub, где хранится Ваша
программа. Или иным удобным для Вас способом.


# Решение:

1. Определяем индексы максимального и минимального элементов.
3. Берем элементы между ними.
4. Суммируем отрицательные элементы.
